Geezers custom WGC Driven Reprojector for the CubeVi Play 5K :D

This is undercooked, and feel free to improve it as you see fit, and I welcome commits to make it nicer!

The benefits:
- WGC! You can use windows that are *not* fullscreen! You can also select WHICH window/screen to use! And lower latency yay!
- Mouse cursor visible toggle, for games which use a software cursor.
- Toggle passthrough! You can put the game behind the reprojection, and it wont lose focus, you can then click through the window and the game gets its! Makes geo11 games with hardware cursor viable! Kerbal space program for example.

Some important notes:
- This is targeting Unity Editor 2021.3.45f2
- This requires installing this plugin made by a legend: https://github.com/ruccho/UnityGraphicsCapture
